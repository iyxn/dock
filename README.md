# Instalasi

Clone repository ini

```bash
  git clone https://github.com/iyxn/gemini-fastapi.git
  cd gemini-fastapi
```
Docker
```bash
  docker-compose --build
```
Akses aplikasi
```bash
  http://127.0.0.1:8501/
```
