# Instalasi

Clone repository ini

```bash
  git clone https://github.com/iyxn/gemini-fastapi.git
  cd gemini-fastapi
```
Edit file .env.backup dan simpan dengan nama file .env
```bash
  API_KEY = API KEY Gemini Anda
```
Build Docker
```bash
  docker-compose --build
```
Akses aplikasi
```bash
  http://127.0.0.1:8501/
```
